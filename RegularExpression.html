RegExp

RegExp is often used with search() and relace().

global match g

multiline search with m

RegExp Patterns

[abc] finds any character between the square brackets

[0-9] finds any digits between the square brackets.

(x | y) finds any of the alternatives separated with |.

Metachrecters

\d finds a digit

\s finds a whitespace character

\b finds a match at the beggining of the word.

test()

It searches a string for a pattern, and returns true or false.
