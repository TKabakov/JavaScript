parseInt() Method

parseInt() parses a string and returns a whole numner. Spaces are allowed. Only the first nember is return.

parseFloat() Method

parseFloat() parses a string and returns number. Spaces are allowed. Only the first number is returned.

Number.isIntegr() Method

Returns true if the argument is integer.

Number.isSafeInteger() Method

A safe integer is an integer that can be exactly repersented as a double precision number.

Safe integres are all integrs fron -(2^53 - 1) to + (2^53 - 1).

Number.parseFloat() Method

Number.parseFloat() works the same way as parseFloat() method.

Number.parseInt() Method
