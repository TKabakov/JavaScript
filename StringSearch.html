String Search

indexOf()

Returns the index of position of the first occurrence of a string in the string.

lastIndexOf()

Returns the index of the last occurrence of a string in a string.

lastIndexOf() searches backwards(from end to the beginning.

search() method

Returns the position of string or regular expression in a string

Allthough search() and indexOf() look very simmilar, there are feww differances:

  • search() cannot take a second start position argument;
  • indexOf() cannot take regular expression as argument.

match()

Returns an array containing the result of matching a string against a string ot regular expression.

If the regular expression does not include the g modifier, the match() will return only the first in the string.
