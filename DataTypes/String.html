JS Strings

You can use quotes inside a string, as long as they don't match the quotes surrouding the string.

JS script length property

The escape sequence \" inserts a double quote in a string.

The escape sequence \' inserts a dingle quote in a string.

The escape sequance \\ inserts a backlash in a string.

You can break a code line within a text string with a backlash.

The safest way to break a code line in a string is using string addition.

You cannot break a code line with a \ backlash, only the string.

JS String as Object

Normally, JS strings are primitive values, but string can aslo be defined as objects with keyword new.

The new keyword complicates the code and slows down execution speed.

Strings and objects cannot be safety compared.
