JS HTML Events

HTML allows event handler attributes, with JS code to be added to HTML element.

In the next example, the code changes the content of its own element

JS code is often several lines long. It is more common to see event attribute calling a function.

Common HTML Events

  • onchange - A HTML element has been changed.
  • onclick - The user clicks an HTML element.
  • onmouseover - The user moves the mouse over an element.
  • onmouseout - The user moves the mouse away from an element.
  • onkeydown - The user pushes the kayboard key.
  • onload - Th browser has finished loading the page.
