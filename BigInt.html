JS BigInt variables are used to store big integer values that are too big t be represented by Number.

JS integers are only accurate up to 15 digits:

To create BigInt thete are two ways:

  • append n to the end;
  • call BigInt with integre in quotes.

Operators that can be used with Number, can also be used on a BigInt.

Arithmetic between a BigInt and a Number is not allowed.
