Tke slice() Method

The slice () extracts a part of a string and returns the extracted part in a new string.

The method takes 2 parameters: start position counting from 0 and end position(optional).

If the second parameter is omitted, the method will slice out the rest of the string:

If a parameter is negative , the position is counted from the end of the string.

JavaScript String substring()

The substring() is similar to slice() with the diffrence that start and end values less that 0 are treted as 0.

JavaScript String substr()

The substr() is similar to slice(), with the difference that the second parameter specifies the length of the extracted part.

If you omit second parameter, substr() will slice out the rest of the string.

When the first parameter is negative, the position counts from the end.
