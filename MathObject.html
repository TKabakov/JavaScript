Math Properies (Consatants)

JS provides 8 mathmatical constants that can be accessed as Math properies:

Math.round()

Math.ceil()

Rounds up to the nearest integer

Math.floor()

Rounds down to the nearest integer

Math.trunc()

Returns the integer part of a number.

Math.sign()

Returns -1 if number is negatice, 1 if it is positive and 0, if it is 0.

Math.pow()

Returns the value of x to the power of y.

Math.sqrt()

Math.abs()

Returns the absolute (positive) value of x.
